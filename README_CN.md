# ANGFAN BRAND SHOP OpenCart Style Static Site

This is a Cloudflare Pages static product shop and copyright record site.

## Main features

- OpenCart-like product detail page
- Product gallery thumbnails
- First Release Time
- Copyright Information
- Qty / Add To Cart visual layout
- Available Options
- Products managed by products.csv
- No visible Brand Rights page
- Copyright owner: LUWANJUN
- Rights contact: ANGFANBRAND163@163.COM

## Upload to GitHub

Upload all files in this folder to the repository root and commit changes.

Cloudflare Pages will redeploy automatically.
